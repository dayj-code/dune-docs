# Blocks

This table contains data about blocks in the Fuel blockchain.

## Table Schema

| Column Name | Data Type | Description |
|------------|-----------|-------------|
| block_number | bigint | The block number |
| block_hash | string | The block hash |
| parent_hash | string | The hash of the parent block |
| timestamp | timestamp | The block timestamp |
| gas_used | bigint | The amount of gas used in the block |
| gas_limit | bigint | The gas limit for the block |
| base_fee | bigint | The base fee for the block |
| transactions_count | bigint | The number of transactions in the block |
