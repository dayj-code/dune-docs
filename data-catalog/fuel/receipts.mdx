# Receipts

This table contains data about transaction receipts in the Fuel blockchain.

## Table Schema

| Column Name | Data Type | Description |
|------------|-----------|-------------|
| transaction_hash | string | The hash of the transaction this receipt belongs to |
| block_number | bigint | The block number containing this receipt |
| block_hash | string | The block hash containing this receipt |
| receipt_index | bigint | The index of the receipt in the block |
| status | string | The status of the transaction |
| gas_used | bigint | The amount of gas used by the transaction |
| logs | array | Array of event logs emitted during transaction execution |
| logs_bloom | string | Bloom filter for the logs |
